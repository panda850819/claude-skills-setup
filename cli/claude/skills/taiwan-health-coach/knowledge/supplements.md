## 重要交互作用

### 必須分開服用

| 組合 | 原因 | 間隔時間 |
|------|------|----------|
| 鈣 + 鐵/鋅 | 鈣會阻擋鐵和鋅的吸收 | 至少 2 小時 |
| 鈣 + 魚油 | 可能形成皂化反應，降低吸收 | 至少 2 小時 |
| B-Complex + 鈣 | B12 吸收可能受影響 | 至少 2 小時 |

### 可以一起服用

| 組合 | 原因 |
|------|------|
| B-Complex + 魚油 + 葉黃素 | 都是早上適合吃的，無交互作用 |
| 鈣 + D3 | D3 幫助鈣吸收（已經在同一顆） |
| Ashwagandha + 鈣 | 無交互作用，都適合睡前 |
| Creatine + 任何 | Creatine 無顯著交互作用 |

## 特別注意事項

### Ashwagandha (南非醉茄)
- 可能降低血壓，如有低血壓請注意
- 可能影響甲狀腺功能，有甲狀腺問題請諮詢醫師
- 建議週期服用（8 週吃、2 週休息）

### Creatine (肌酸)
- 多喝水（每日 2.5-3L），避免脫水
- 可能造成初期體重上升 1-2kg（水分滯留，正常現象）
- 粉末可加入咖啡、蛋白粉、水中

### Calcium + D3
- 不要與鐵劑、鋅、抗生素同時服用
- 單次不超過 600mg 鈣（吸收上限）

### Fish Oil
- 手術前 1-2 週停用（影響凝血）
- 冷藏保存，避免氧化

## 減脂期補充品優先順序

根據你的目標（84kg → 65kg），優先順序：

1. **必須** - Fish Oil (抗發炎、心血管)
2. **必須** - Calcium + D3 (減脂易流失鈣)
3. **推薦** - B-Complex (代謝支援)
4. **推薦** - Creatine (保留肌肉)
5. **可選** - Ashwagandha (壓力管理)
6. **可選** - Lutein (護眼)
7. **可選** - Black Sesame (營養補充)
