# 手掌份量估算法

用戶使用手掌法則記錄飲食，以下是換算標準。

---

## 手勢對照表

| 手勢 | 代號 | 重量 | 容量 | 說明 |
|------|------|------|------|------|
| 👊 拳頭 | 1拳 | ~150g | ~240ml (1碗) | 飯、麵、蔬菜、水果 |
| 👊½ 半拳 | 半拳 | ~75g | ~120ml (半碗) | 減脂期飯量 |
| 🖐️ 手掌 | 1掌 | ~100g | - | 肉類（厚度約1指） |
| 🖐️½ 半掌 | 半掌 | ~50g | - | 小份肉類 |
| 🤲 掌心凹 | 1凹 | ~30g | - | 堅果、果乾 |
| 👍 拇指 | 1指 | ~15g | ~15ml | 油脂、醬料 |

---

## 常見表達方式

用戶可能這樣說：

| 表達方式 | 理解為 |
|---------|--------|
| 「一掌雞胸」 | 100g 雞胸肉 |
| 「一個拳頭的飯」 | 1碗飯 160g |
| 「半拳飯」 | 半碗飯 80g |
| 「兩拳青菜」 | 300g 蔬菜 |
| 「一掌半的肉」 | 150g 肉類 |
| 「一個大拇指的油」 | 15ml 油 |

---

## 減脂期每餐建議份量

| 食物類型 | 建議份量 | 熱量佔比 |
|---------|---------|---------|
| 蛋白質 | 1-1.5 掌 | ~150-200 kcal |
| 蔬菜 | 2 拳 | ~50-70 kcal |
| 碳水 | 半拳-1拳 | ~140-280 kcal |
| 油脂 | 0.5-1 指 | ~70-135 kcal |
| **每餐總計** | - | **400-600 kcal** |

---

## 一日三餐範例（減脂期 1,600 kcal）

### 早餐 (~400 kcal)
- 燕麥 1拳 (180 kcal)
- 雞蛋 1顆 (80 kcal)
- 藍莓 半拳 (43 kcal)
- 黑咖啡 (5 kcal)
- 牛奶少許 (50 kcal)

### 午餐 (~550 kcal)
- 雞胸肉 1.5掌 (176 kcal)
- 白飯 半拳 (140 kcal)
- 燙青菜 2拳 (70 kcal)
- 油脂 1指 (135 kcal)

### 晚餐 (~500 kcal)
- 牛肉片 1掌 (180 kcal)
- 燕麥/地瓜 半拳 (90 kcal)
- 燙青菜 2拳 (70 kcal)
- 奇異果 1顆 (56 kcal)
- 油脂 0.5指 (70 kcal)

### 點心 (~150 kcal)
- 堅果 1凹 (180 kcal) 或
- 水果 1拳 (80 kcal) + 無糖豆漿

---

## 快速換算公式

```
蛋白質：1掌 ≈ 100g ≈ 20-25g 蛋白質
碳水：1拳 ≈ 1碗 ≈ 50-60g 碳水
蔬菜：1拳 ≈ 150g ≈ 30-40 kcal
油脂：1指 ≈ 15ml ≈ 135 kcal
```
