# Gemini CLI Setup

## 安裝

```bash
npm install -g @google/generative-ai-cli
```

或使用 Homebrew:

```bash
brew install gemini-cli
```

## 設定 API Key

```bash
export GEMINI_API_KEY="your-api-key"
```

建議加入 `~/.zshrc` 或 `~/.bashrc`。

## 驗證安裝

```bash
gemini --version
```
